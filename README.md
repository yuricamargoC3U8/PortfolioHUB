## 📌 O que é PortfolioHUB?

**PortfolioHUB** é um centro integrado de projetos, criado para **apresentar, organizar e compartilhar** meus trabalhos acadêmicos, pessoais e profissionais em um único lugar. Um espaço onde **linguagens de programação, tecnologias e criatividade** se encontram.

## 🚀 Como Começar

### Clonar o Repositório

```bash
git clone https://github.com/yuricamargoC3U8/PortfolioHUB.git
cd PortfolioHUB
```

### Explorar Localmente

```bash
# Abrir index.html em seu navegador
open index.html

# Ou, se estiver no Linux
xdg-open index.html

# Ou no Windows
start index.html
```

### Navegar pelos Projetos

```bash
# Projetos em Python
ls projetos-academicos/Python\ Projects/

# Projetos em Go
ls projetos-academicos/Golang\ Projects/

# Documentação
cat docs/README.md
```
